# Nicolas Diacono — Digital Portfolio

A modern, responsive portfolio website showcasing my experience as a Systems Technician and Qualified Electrician based in Christchurch, New Zealand.

## [View Live Site →](https://nidiaco.github.io/CV/)

---

## About

This portfolio highlights my professional journey across the electrical and solar energy industry — from apprentice electrician to microinverter engineering at Enphase Energy. It includes my work experience timeline, technical skills, qualifications, and contact details.

## Features

- **Responsive Design** — Looks great on desktop, tablet, and mobile
- **Animated Hero** — Typewriter effect, floating particles, and cursor glow
- **Interactive Timeline** — Expandable work experience cards with personal insights
- **Skills Section** — Animated progress bars and hoverable technology tags
- **Animated Counters** — Stats count up on scroll
- **Quotes Carousel** — Rotating thoughts on renewable energy
- **Record of Achievement** — Downloadable NZ qualifications PDF

## Tech Stack

- HTML5 / CSS3 / JavaScript (Vanilla)
- Bootstrap 5.3
- Bootstrap Icons
- Google Fonts (Inter, Playfair Display)
